This is the second tutorial on creating historgrams via RStudio. Install the following and pull from the library:

library(dplyr)
library(Lahman)
library(ggplot2)
library(ggiraph)

Extracting the Data

In the last post, we assigned our extracted data to df(dataframe).Complete this step again.

df<-Teams%>%
  filter(yearID==1980)%>%
  select(name,HR)%>%
  arrange(HR)

str give the structure of whatever data you have. Store your data as a factor vector, rather than a character vector.The benefit to this is that factor vectors are organized into different levels . Type class(df\(name)</b> to confirm that the data has been stored correctly. Use <b>levels(df\)name) to view levels. Use levels=df$name to set levels to the same order as the columns.

str(df$name)  
##  chr [1:26] "New York Mets" "San Diego Padres" "Houston Astros" ...
df$name<-factor(df$name,levels=df$name)

Data visualization

In the last post, we used ggplot to create a visual representation of the data. In this representation, we are going to name our x and y coordinates with xlab and ylab, repsectively.

g<-ggplot()+
  geom_bar(data=df,aes(x=name,y=HR,tooltip=HR, data_id=name), stat="identity", color="blue",fill="white")+
  coord_flip()+
  ggtitle("1980 Team Homeruns")+
  xlab("Team")+
  ylab("Homeruns")
## Warning: Ignoring unknown aesthetics: tooltip, data_id

Optional: Use ggiraph to change color characteristics when hovering over the graph:

ggiraph(code=print(g), hover_css=fill:“red”;stroke:“white”)
