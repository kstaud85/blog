This is the second tutorial on creating historgrams via RStudio. Install the following and pull from the library:

library(dplyr)
library(Lahman)
library(ggplot2)

Extracting the Data

Assigned the extracted data to result (dataframe).

result<-Batting%>%
        filter(playerID=='ruthba01')%>%
        select(yearID,SO,HR)

Data visualization

In the last post, we used ggplot to create a visual representation of the data. In this representation, we are going to name our x and y coordinates with xlab and ylab, repsectively. Use ggiraph to change color characteristics when hovering over the graph.

ggplot()+
  geom_histogram(data=result,aes(x=HR),bins=5,color="red",fill="white")
